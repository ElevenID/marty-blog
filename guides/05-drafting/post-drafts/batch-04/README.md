## Why this batch belongs together

This batch makes governance and deployment proof concrete in two regulated operational environments:

- first, show how standards and rule changes become explicit compliance-profile updates rather than backend code churn
- second, prove the model in a high-throughput travel lane where trust, caching, and policy windows matter
- third, prove the model again inside enterprise access where one credential can support badge, network, and app decisions under governed Cedar policy

Together, these drafts demonstrate that Marty's policy and profile surfaces are not theoretical abstractions. They are the operating model.
