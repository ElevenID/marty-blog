## Why this batch belongs together

This batch closes the remaining ecosystem and implementation proof cluster by doing three related jobs:

- first, prove that Marty matters beyond government identity by showing a non-state membership entitlement that can travel across organizations under shared trust governance
- second, show that OID4VCI fits Marty as an issuance transport surface rather than replacing the object model underneath
- third, show that OID4VP carries a presentation exchange while trust, disclosure, and deployment remain Marty decisions

Together, these drafts make the architecture feel both broader and more buildable. The membership proof widens the field of use. The OID4VCI and OID4VP notes then tell implementers exactly where the transport standards attach to the Marty model without collapsing the model into protocol plumbing.
